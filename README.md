## Installation

1.
```bash
brew tap laisbastosbg/taskinator
```

2.
```bash
brew install taskinator
```

## Usage

- See tasks

```swift
taskinator
```

- Add new task

```swift
taskinator -a <task description>
```

- Complete task

```swift
taskinator -f <task index>
```

- Delete new task

```swift
taskinator -d <task index>
```
